## 필요 데이터

`로그인`

기초 데이터 [엑셀데이터]
음식 : [종류][타입(SP,친밀도)][수치]
상호작용: [타입(SP,친밀도)][수치]

`위치` - 데이터 로드 위치지정(기본값)
현재 GPS경로위치

`스타일변경` - 데이터 로드 스타일 지정 (기본값)
스타일 -> 프리팹
